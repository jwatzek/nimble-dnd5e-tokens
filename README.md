# Nimble - DND 5e Tokens

A simple module that maps the tokens from the [Dungeons & Dragons Fifth Edition](https://foundryvtt.com/packages/dnd5e) system to the Nimble System built-in actors. The `dnd5e` system needs to be installed on your Foundry instance, so that the token images are available.

In simple words, it allows you to use the D&D monster tokens with Nimble.

## Installation

1. On Foundry VTT's **Configuration and Setup** screen, go to **Add-on Modules**
1. Click **Install Module**
2. In the Manifest URL field, paste: `https://github.com/jwatzek/nimble-dnd5e-tokens/releases/latest/download/module.json`
3. Click **Install** next to the pasted Manifest URL
